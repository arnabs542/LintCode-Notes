## O(1) Check Power of 2  ##

Using O(1) time to check whether an integer n is a power of 2.

Example

	For n=4, return true;
	
	For n=5, return false;

    bool checkPowerOf2(int n) {
        // write your code here
        if (n <= 0) {
            return false;
        }
        return !(n & (n - 1));
    }
